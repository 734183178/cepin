# CEPIN 心理测评平台

一个集成了多种专业心理测试的综合测评网站，提供免费、科学的在线心理评估服务。

## 🎯 项目概述

CEPIN是一个多功能心理测评平台，包含多种精心设计的心理测试，帮助用户更好地了解自己的性格特质、关系模式和心理状态。

## 📋 包含的测试项目

### 🧠 Love Brain 恋爱脑测试
- **路径**: `/lovestruck/`
- **功能**: 评估恋爱模式和依恋风格
- **特色**: 32道科学题目，6维度分析，即时结果
- **优化**: SEO完善、响应式设计、首屏测试入口

### 🍎 水果性格测试
- **路径**: `/fruit/`
- **功能**: 通过水果偏好分析性格特征
- **特色**: 趣味性强，结果生动形象

### 👨‍👩‍👧‍👦 家庭关系测试
- **路径**: `/family/`
- **功能**: 评估家庭关系和亲密度
- **特色**: 支持中英文版本，多维度评估

### 🌟 其他测试项目
- **动物性格测试** (`/animal/`)
- **黑暗人格测试** (`/Darkpersonality/`)
- **心理年龄测试** (`/Psychologicalage/`)
- **才能测试** (`/talent/`)
- **SCL-90心理健康测评** (`/scl-90zice/`)
- **社会关系影响力测试** (`/sri-test/`)
- **RPI人格测试** (`/rpi/`)

## 🚀 快速开始

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/734183178/cepin.git

# 进入项目目录
cd cepin

# 使用任意HTTP服务器运行
# 例如使用Python:
python -m http.server 8000

# 或者使用Node.js的http-server
npx http-server

# 访问 http://localhost:8000
```
